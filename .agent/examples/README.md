# AI Agent Examples

This directory contains reference images and examples for AI coding agents to understand the project's UI/UX patterns and design language.

## Purpose

When working on UI components or features, AI agents can reference these examples to:

- Understand the visual design system
- Maintain consistency with existing UI patterns
- See examples of successful implementations

## Naming Convention

Use descriptive names instead of timestamps:

- ✅ `example-ui-layout.png`
- ✅ `reference-color-palette.png`
- ✅ `example-tool-interface.png`
- ❌ `Screenshot 2026-01-08 at 7.10.47 AM.png`
